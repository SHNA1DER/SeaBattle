1) tkinter:
Эта библиотека предоставляет интерфейс для создания графических пользовательских интерфейсов (GUI) в Python. Она основана на библиотеке Tk, которая является стандартной для языка программирования Tcl/Tk.
from tkinter import *: Импортирует все имена из модуля tkinter. Это включает классы и функции для создания окон, виджетов (кнопок, полей ввода, меток и т.д.) и управления событиями.
from tkinter import messagebox: Импортирует модуль messagebox, который предоставляет функции для отображения различных типов сообщений и диалогов, таких как информационные окна, предупреждения и запросы подтверждения.
2) time:
Библиотека time предоставляет различные функции для работы со временем. Основные возможности включают получение текущего времени, задержку выполнения программы, вычисление временных интервалов и многое другое.
Примеры функций: time.sleep(), time.time() и другие.
3) random:
Модуль random предназначен для генерации псевдослучайных чисел. Он содержит множество функций для случайного выбора элементов из списка, генерации целых чисел, вещественных чисел и других операций.
Примеры функций: random.randint(), random.choice(), random.random() и другие.
